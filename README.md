# Di Barbuda

Projeto de jogo digital desenvolvido utilizando **Unity** e **C#** como parte do curso **Técnico em Programação de Jogos Digitais**.

## Sobre o projeto

O objetivo deste projeto foi aplicar conceitos de programação orientada a objetos, desenvolvimento de jogos e criação de aplicações interativas utilizando a engine Unity.

Durante o desenvolvimento foram implementadas mecânicas de jogo, scripts em C#, gerenciamento de cenas e integração de recursos gráficos.

Embora tenha sido desenvolvido em 2012/2013, este projeto representa um marco importante na minha trajetória com desenvolvimento de software e faz parte do meu portfólio técnico.

## Tecnologias utilizadas

- Unity
- C#
- Visual Studio
- Blender

## Compatibilidade

A versão compilada disponível na seção **Releases** é compatível **apenas com o sistema operacional Windows**.

## Funcionalidades

- Mecânicas de gameplay
- Programação em C#
- Gerenciamento de cenas
- Organização de assets
- Integração de modelos 3D
- Testes e depuração

## Como executar

### Opção 1 – Jogar

Baixe a versão compilada disponível na seção **Releases**, extraia o arquivo `.zip` e execute o arquivo **Di Barbuda.exe**.

> **Observação:** A versão disponibilizada é compatível apenas com **Windows**.

### Opção 2 – Abrir o projeto

1. Clone este repositório.
2. Abra o projeto utilizando uma versão compatível do Unity.
3. Aguarde a importação dos assets.
4. Execute o projeto pelo Editor da Unity.
